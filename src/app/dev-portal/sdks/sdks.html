PayMo Python SDK

Python 3.7+ support, synchronous and async variations included.

pip install paymo-python
from paymo import PayMoClient;

paymo = PayMoClient(api_key="sk_test_12345");

transfer = paymo.disbursements.create(amount=50000, currency="KES", destination=paymo.Destination.bank_account(bank_code="068", account_number="01234567890"), purpose="Salary payment");
print(transfer.status);
PayMo PHP SDK

PHP 7.4+ and 8.x compatible. Laravel package also available.

composer require paymo/paymo-php
$paymo = new \PayMo\PayMoClient('sk_test_12345');

$invoice = $paymo->invoices->create([
'customer' => ['name' => 'John Doe', 'email' => 'john@example.com'],
'items' => [['name' => 'Consulting', 'amount' => 10000]],
'due_date' => '2025-07-01'
]);
echo $invoice->payment_url;
PayMo Flutter Plugin

Drop-in payment UI for iOS and Android apps.

flutter pub add paymo_flutter
import 'package:paymo_flutter/paymo_flutter.dart';

await PayMoCheckout.start(
context,
config: PayMoConfig(publicKey: 'pk_test_12345', amount: 1500, currency: 'KES', customerEmail: 'user@app.com'),
onSuccess: (result) => print('Success: ' + result.reference),
onError: (error) => print('Error: ' + error.message),
);

from paymo.async_client import AsyncPayMo
