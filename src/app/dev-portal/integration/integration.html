4.7.1 — Integration Patterns

Select and configure how your application frontend and backend interface with PayMo.

Direct API

RESTful server-to-server calls. Includes webhook listeners, idempotency, and fallback polling.

Embedded Checkout

iFrame, Popups, or Redirects. Reduces PCI scope while maintaining UX.

White-Label

Fully branded payment pages, emails, SMS, and custom domains mapping.

Hybrid Setup

Mix API + SDK. Offline queueing, bulk background processing.
