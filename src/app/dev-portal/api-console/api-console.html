Get started quickly with PayMo SDKs.

curl https://api.paymo.com/v1/charges \ -H "Authorization: Bearer sk_test_8f92..." \ -H "Content-Type: application/json" \ -d '{ "amount": 5000, "currency": "KES", "source": "mpesa_stk", "phone": "0712345678" }'
const paymo = require('paymo')('sk_test_8f92...'); const charge = await paymo.charges.create({ amount: 5000, currency: 'KES', source: 'mpesa_stk', phone: '0712345678' });
